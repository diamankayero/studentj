# Markdown Basic Syntax

QWER has markdown parser and compiler built-in. You can write posts with Markdown Syntax.

If you already familar with Markdown Basic Syntax, you can skip the rest.

## Headers

```md
# h1

## h2

### h3

#### h4

##### h5

###### h6
```

## Bold

```md
**Bold Text**
```

**Bold Text**

## Italic

```md
_italicized text_
```

_italicized text_

## Cross Out Text

```md
~~Cross Out Text~~
```

~~Cross Out Text~~

## Blockquote

```md
> blockquote
```

> blockquote

## Ordered List

```md
1. First item
2. Second item
3. Third item
```

1. First item
2. Second item
3. Third item

## Unordered List

```md
- First item
- Second item
- Third item
```

- First item
- Second item
- Third item

## Inline Code

```md
`Inline Code`
```

`Inline Code`

## Horizontal Rule

```md
---
```

---

## Link

```md
[QWER](https://svelte-qwer.vercel.app/)
```
